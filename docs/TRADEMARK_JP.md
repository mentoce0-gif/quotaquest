# TRADEMARK_JP.md — JPO（日本）商標出願 準備パッケージ

> 私（AI）は出願代理・印紙代支払いはできません。本書は**出願直前まで**の準備一式です。
> 最終の「願書提出・料金納付」は**ご本人または登録弁理士**が行います。金額・区分の最終確認は
> 必ず[JPO料金ページ](https://www.jpo.go.jp/system/process/tesuryo/hyou.html)と弁理士で。

## 0. 前提
- 出願商標（標章）：**QuotaQuest**（標準文字＝文字商標で出願を推奨）
- 出願人：未定（個人 or 法人）→ §4で記入
- 日本は**先願主義**（早い者勝ち）。名称確定後はできるだけ早く出願。

## 1. 区分（Nice分類）と指定商品・役務（文案）
ソフト＋SaaSの定石は **第9類＋第42類** の2区分。予算重視なら第9類のみでも可。

### 第9類（ダウンロード可能なソフトウェア）
```
ダウンロード可能なコンピュータソフトウェア；
コンピュータプログラム；
ダウンロード可能なコンピュータアプリケーションソフトウェア；
電子計算機の利用状況・使用量を監視し可視化するためのコンピュータソフトウェア；
コンピュータ用ウィジェット（ダウンロード可能なもの）
```

### 第42類（SaaS・ソフトウェア開発）※2区分目（任意）
```
ソフトウェアとしてのサービス（SaaS）の提供；
電子計算機用プログラムの提供；
コンピュータソフトウェアの設計・開発・保守；
クラウドコンピューティングに関する役務
```

> 指定は広すぎると拒絶・費用増、狭すぎると保護不足。上記は標準的な落とし所。弁理士に微調整可。

## 2. 費用の目安（特許庁費用／要・最新確認）
| 項目 | 計算式 | 1区分 | 2区分 |
|------|--------|-------|-------|
| 出願料 | 3,400 +（8,600 × 区分数） | 12,000円 | 20,600円 |
| 設定登録料（5年分割） | 約16,400〜17,200 × 区分数 | 約16,400〜17,200円 | 約32,800〜34,400円 |
| 設定登録料（10年一括） | 約28,200〜32,900 × 区分数 | 約28,200〜32,900円 | 約56,400〜65,800円 |

- **最小ライン（1区分・5年）＝ 出願12,000＋登録約16,400〜17,200 ≒ 約2.9〜3.5万円**。
- ※登録料は改定履歴があり数値に幅があるため、**JPO公式で必ず最新額を確認**。
- 弁理士に依頼する場合は上記＋**代理人手数料 概ね5〜15万円/件**（事務所による）。
- オンライン商標サービス（例：Cotobox/Toreru等）は中間的価格帯。

## 3. 出願ルート（どれか1つ）
1. **自力・電子出願**：特許庁の電子出願ソフト＋識別番号・電子証明書が必要。最安だが手間大。
2. **自力・書面出願**：紙の願書を郵送/持参。別途**電子化手数料**（書面→電子化）が発生。
3. **オンラインサービス**：Web上で区分選択→簡易調査→出願代行。中価格・手軽。
4. **弁理士**：調査・区分設計・中間対応まで任せられる。最も安全（推奨：本番）。

## 5. 出願前チェック（拒絶を避ける）
- [ ] **J-PlatPat で全文・類似検索**（同一/類似区分に先行登録がないか）※簡易Web検索は不十分
- [ ] 識別力の確認（記述的・一般的でないか）→ QuotaQuestは造語性ありで比較的良好
- [ ] 商品/役務が事業実態（ソフト配布＋SaaS）と整合
- [ ] ロゴも保護したい場合は**文字商標とは別に図形/結合商標を別出願**するか検討
- [ ] 米国等も視野なら**マドプロ（国際登録）**を弁理士と相談（日本出願を基礎に）

## 6. 審査の流れ・期間
1. 出願 → 方式審査 → 実体審査（**標準で概ね6〜12か月**。早期審査制度の利用で短縮可）。
2. 拒絶理由通知が来たら**意見書/補正書**で対応（弁理士推奨）。
3. 登録査定 → **登録料納付** → 設定登録 → 公報掲載。権利は登録から**10年**（更新可）。
