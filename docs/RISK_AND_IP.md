> 本書は一般的情報であり法的助言ではありません。出願・係争の最終判断は弁理士/弁護士へ。
> 結論：このプロダクトで最も現実的な“取り下げ”リスクは「自分が名前を先取りされる」側
> ではなく、**自分が他者の商標（特に「Claude」）を侵害してしまう**側。順番を間違えないこと。

## 要点
- **(A) コードは「未登録だから奪われる」ものではない。** 著作物は自動的に作者のもの。模倣品が
  商標を取っても、あなたのコードを取り上げる権利は生じない。公開＝防御に働く。
- **(B) 売れなくなる最有力シナリオは“加害者”側＝「Claude」ブランド名。** 商標登録では防げず、
  改名・記述的表記・ディスクレーマーでのみ回避可能。→ **改名を出願より先に**実施済。

## 優先アクション（実施状況）
1. ✅ 改名（`QuotaQuest`）— Claudeをブランド名から除去
2. ✅ 表記整備 — 「QuotaQuest — usage HUD for Claude Code」＋「Not affiliated with Anthropic」
3. ✅ ライセンス衛生 — `NOTICE` / `THIRD-PARTY.md` / `LICENSING.md`
4. ⬜ 早期公開 — GitHubへpush（公開日＝先使用の証拠）
5. ⬜ JPO出願 — 名称確定後すぐ（`TRADEMARK_JP.md` の様式で）

## 万一「停止通知（C&D）」が来たら
- 慌てて削除も無視もしない。**発信者・主張根拠（どの登録番号/権利か）を確認**。
- 名称問題なら**改名で即解決**できることが多い（コードは残せる）。
- ライセンス問題なら該当箇所を是正。**弁理士/弁護士に相談**してから返信。
- GitHub/Gumroadの**DMCA/異議申立**手続きの存在を把握しておく。
