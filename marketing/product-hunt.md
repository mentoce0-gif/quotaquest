# Product Hunt launch

**Name:** QuotaQuest

**Tagline:** Turn your AI quota into an MMORPG status bar

**Topics:** Developer Tools · Artificial Intelligence · Productivity · Open Source

---

**Description (the one-liner under the title):**
HP, MP, EXP and a recovery timer for your Claude Code usage — an always-on-top, half-transparent desktop HUD. Stop reading /usage. Start watching your HP.

---

**First comment (maker's intro):**

Hey Product Hunt 👋

I'm a heavy Claude Code user, and I kept hitting the weekly limit mid-task. Claude runs *two* clocks at once — a 5-hour rolling window and a weekly cap — and shows you neither while you're working.

The existing monitors are accurate but look like spreadsheets, so checking your quota feels like a chore. I wanted the opposite: something I'd actually *enjoy* glancing at.

**QuotaQuest** reads the usage logs Claude Code already writes locally and renders them as a game HUD:

- ❤️ **HP** — weekly budget remaining
- 🔵 **MP** — current 5-hour session window
- ⭐ **EXP** — today's usage
- ⟳ **Recovery timer** — when your window frees up
- 🔥 **Burn rate** — LOW / MODERATE / HIGH / EXTREME
- 🟢 **Status** — NORMAL / WARNING / CRITICAL recolors the HP bar

It's **100% local** (no account, no cloud), lightweight, always-on-top and half-transparent — great on a second monitor or in an OBS scene. The core is **open-source (MIT)**; Pro themes and a multi-tool pack (Cursor/Copilot/Windsurf) fund development.

Windows + Rainmeter today; cross-platform is on the roadmap.

Would love your feedback — especially on which themes to build next and which tools to add adapters for. Thanks for checking it out! ⚡
