FAQ

Questions, answered

Does this send my data anywhere?

No. It reads the usage logs Claude Code already writes locally (under ~/.claude) and writes one small state.json on your machine. Nothing leaves your computer — no account, no cloud, no telemetry.

How does it know my limits?

It aggregates your token usage over the 5-hour and 7-day windows and compares against your plan's caps in config.json. Anthropic's exact numbers aren't public, so the caps are sensible defaults you can calibrate to your real plan in one edit.

Do I need Rainmeter?

Yes, for the desktop HUD on Windows — it's a free, lightweight customization tool. The installer drops the skin straight into your Rainmeter folder. A cross-platform Electron/Tauri build is on the roadmap.

Will it work with Cursor / Copilot / Windsurf?

Claude Code is supported today. Cursor, Copilot and Windsurf adapters ship in the Multi-Tool pack — they all expose depletable budgets, so the same HP/MP model applies.

Is it really free?

The core HUD is open-source (MIT). Pro themes, animated status effects, and the multi-tool/streamer packs are paid and fund ongoing development.

Will it slow down my machine?

No. The monitor polls on an interval (30s by default) and the skin reads one local file. Idle CPU is essentially zero and memory use is tiny.

I'm a streamer — can I put this on screen?

That's a core use case. It's always-on-top and half-transparent, so it drops into OBS as-is. The Multi-Tool pack adds a transparent-background source built for streams.
