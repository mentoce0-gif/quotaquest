# UI_WIREFRAME.md — AI Energy HUD

> Phase 3 / UI ワイヤーフレーム
> 最終更新: 2026-06-23

MVP の既定テーマは **サイバーパンク HUD**。他テーマ（FF風/ミニマル/ターミナル/ゆるキャラ/円形）は同 state.json を共有する皮替えとしてロードマップ化。

---

## 1. メイン HUD（既定・縦約 240px / 横約 360px）

```
┌──────────────────────────────────────────────┐
│ ◆ CLAUDE CODE                      ● ONLINE   │
│   AI ENERGY HUD v1.0                           │
│                                                │
│  HP ▰▰▰▰▰▰▰▰▰▰▰▰▰▱▱   87%                      │
│  ♥  425,362 / 500,000 tokens                   │
│                                                │
│  MP ▰▰▰▰▰▰▰▰▰▱▱▱▱▱▱   63%                      │
│  ⏻  3.1h / 5h                                  │
│                                                │
│  ↻ REGEN  +2,140/min   ⏳ RECOVERY 01:52:18    │
│  🔥 BURN  -12.4k/h      ◇ STATUS  NORMAL       │
│                                                │
│  TODAY 1.82M   WEEK 8.73M   MONTH 32.41M       │
└──────────────────────────────────────────────┘
```

### 構成要素
- **タイトル行**: ロゴ + プロダクト名 + オンライン状態ドット（緑）。
- **HP バー**: 横長グラデーション（緑→黄→赤を残量で動的）。右に %、下に `current / max tokens`。
- **MP バー**: 青系グラデーション。下にセッション経過 `xh / 5h`。
- **ステータス行**: REGEN / RECOVERY / BURN / STATUS を2×2で。
- **EXP 行**: TODAY / WEEK / MONTH の累計。

### カラー状態（HP%連動）
| 状態 | HP% | バー色 | STATUS文字色 |
|---|---|---|---|
| NORMAL | > 40 | #39FF14 (緑) | 緑 |
| WARNING | 15–40 | #FFD400 (黄) | 黄 |
| CRITICAL | < 15 | #FF3B3B (赤・点滅) | 赤 |

## 2. パーティ HP（マルチエージェント・展開行）

```
┌──────────────────────────────────────────────┐
│  PARTY                                         │
│  🟦 CLAUDE  ▰▰▰▰▰▰▰▰▱▱  87%  NORMAL            │
│  🟩 CODEX   ▰▰▰▰▰▱▱▱▱▱  54%  WARNING           │
│  ⬜ CURSOR  ▱▱▱▱▱▱▱▱▱▱   --  (manual)          │
└──────────────────────────────────────────────┘
```

- 各エージェントを1行 = RPG のパーティメンバー。
- Codex がトークン不明（turn_context 欠落）なら `?? UNKNOWN` をオレンジで表示し **誤った安心を与えない**。
- メインゲージは `primary_agent` を拡大表示、下にパーティ全員。

---

## 3. コンパクト / 配信モード

```
┌───────────────────────────────┐
│ HP ▰▰▰▰▰▰▰▰▱▱ 87%  MP ▰▰▰▱ 63% │   ← OBS オーバーレイ向け1行
└───────────────────────────────┘
```

- 高さ最小・背景透明度を上げ、配信画面の隅に常駐。
- ドラッグで自由配置、右クリックでテーマ/モード切替（Rainmeter コンテキストメニュー）。

---

## 4. インタラクション
- **ドラッグ**: HUD 移動（位置は記憶）。
- **右クリック**: Rainmeter メニュー → Variant 切替（メイン/コンパクト/パーティ）、半透明度、最前面 ON/OFF。
- **クリック(タイトル)**: monitor の最終更新時刻ツールチップ（鮮度確認）。

---

## 5. レイアウト指標（実装目安）
- 既定キャンバス: 360×240（メイン）/ 360×320（パーティ展開）/ 320×40（コンパクト）。
- HP/MP バー: 高さ 18px、角丸、内側にグラデ + 外周グロー。
- フォント: 見出し = 角張ったテック系（同梱フォント or システム Segoe UI Semibold 代替）、数値 = 等幅。
- 余白: 12px グリッド。
